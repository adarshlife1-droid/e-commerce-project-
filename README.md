# e-commerce-project-
# Project Overview
This project is a comprehensive data analysis of the Brazilian e-commerce marketplace Olist Store Dataset. The objective is to analyze sales performance, customer behavior, delivery efficiency, seller performance, and overall business trends using real-world transactional data.
The dataset contains detailed information about orders, customers, products, payments, reviews, sellers, and geolocation data. Through this project, we aim to extract meaningful business insights that can support data-driven decision-making.
# Dataset Information
The dataset used in this project is the Olist Brazilian E-Commerce Public Dataset from Kaggle.
It includes the following tables:
Customers
Orders
Order Items
Products
Sellers
Payments
Reviews
Geolocation
Product Category Translation
The data covers multiple years of transactions across Brazil.
# Project Objectives
The main goals of this project are:
Analyze overall sales performance

Analyze delivery times and delays

Measure customer satisfaction through reviews
